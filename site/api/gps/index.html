    Gram Panchayats API¶

    GP-level geospatial data and lookups. Gram Panchayats (GPs) are village-level administrative units within blocks — the finest spatial resolution in the LEAF DSS data hierarchy.

    Data Availability

    GP-level data is currently available for Tinsukia district only (~95 GPs across 8 blocks). For all other districts, use the Blocks API for block-level data. Use GET /api/levels to check which districts support GP data.

    GP data includes variables from the DSS metadata (same codes as block-level: AD, WA, etc.) plus GP-specific indicators for livestock density, transport connectivity, finance, utilities, and agricultural activities.

    Get All GPs¶

    Returns all GPs as a GeoJSON FeatureCollection with all variable data as properties.

    GET /api/gp
GET /api/gp/geojson

    Both paths return identical data.

    Response¶

    {
  "type": "FeatureCollection",
  "features": [
    {
      "type": "Feature",
      "properties": {
        "GP_CODE": 12345,
        "GP_NAME": "Rongagora",
        "Block_Name": "Digboi",
        "Dist_Name": "Tinsukia",
        "VIL_COUNT": 8,
        "AD": 42.1,
        "BW": 8
      },
      "geometry": {
        "type": "Polygon",
        "coordinates": [[[95.4, 27.4], ...]]
      }
    }
  ]
}

    Key Property Fields¶

    Field Type Description
    GP_CODE integer Unique GP identifier
    GP_NAME string Gram Panchayat name
    Block_Name string Parent block name
    Dist_Name string District name (currently always "Tinsukia")
    VIL_COUNT integer Number of villages in this GP
    Variable codes number Indicator values — see GP Variables for metadata

    Errors¶

    Code Description
    404 GP shapefile not available

    Example¶

    curl https://iwmi-leaf.onrender.com/api/gp

    import requests
import geopandas as gpd

r = requests.get("https://iwmi-leaf.onrender.com/api/gp")
geojson = r.json()
print(f"Total GPs: {len(geojson['features'])}")

gdf = gpd.GeoDataFrame.from_features(geojson['features'])
print(gdf[['GP_NAME', 'Block_Name', 'VIL_COUNT']].head(10))

    const r = await fetch('/api/gp');
const geojson = await r.json();
console.log(`Total GPs: ${geojson.features.length}`);

// Add to Leaflet map
L.geoJSON(geojson, {
  onEachFeature: (feature, layer) => {
    layer.bindTooltip(feature.properties.GP_NAME);
  }
}).addTo(map);

    Get GP by ID¶

    Returns a single GP's GeoJSON by its GP_CODE.

    GET /api/gp/{gp_id}

    Parameters¶

    Parameter In Type Required Description
    gp_id path string Yes The GP_CODE identifier (numeric)

    Response¶

    A GeoJSON FeatureCollection with the single matching GP.

    Errors¶

    Code Description
    404 No GP found with the given code

    Example¶

    curl https://iwmi-leaf.onrender.com/api/gp/12345

    Get GP by Name¶

    Returns a single GP's GeoJSON by its GP_NAME.

    GET /api/gp/by-name/{gp_name}

    Parameters¶

    Parameter In Type Required Description
    gp_name path string Yes GP name, case-sensitive

    Response¶

    A GeoJSON FeatureCollection for the matching GP.

    Errors¶

    Code Description
    404 GP not found or GP data not available

    Example¶

    curl https://iwmi-leaf.onrender.com/api/gp/by-name/Rongagora

    import requests

name = "Rongagora"
r = requests.get(f"https://iwmi-leaf.onrender.com/api/gp/by-name/{name}")
if r.status_code == 200:
    gp = r.json()
    props = gp['features'][0]['properties']
    print(f"GP: {props['GP_NAME']}, Block: {props['Block_Name']}")
    print(f"Villages: {props.get('VIL_COUNT', 'N/A')}")

    Get GPs by Block¶

    Returns all GPs belonging to a specific block, with name, code, and village count.

    GET /api/gp/block/{block_name}

    Parameters¶

    Parameter In Type Required Description
    block_name path string Yes Block name, case-sensitive (e.g. "Digboi")

    Response¶

    {
  "block": "Digboi",
  "district": "Tinsukia",
  "gps": [
    {
      "name": "Rongagora",
      "code": "12345",
      "village_count": 8
    },
    {
      "name": "Panitola",
      "code": "12346",
      "village_count": 12
    }
  ]
}
    Field Type Description
    block string Block name
    district string District name
    gps array GPs in this block, sorted alphabetically
    gps[].name string GP name
    gps[].code string GP_CODE identifier
    gps[].village_count integer Number of villages

    Errors¶

    Code Description
    404 No GPs found in block or GP data not available

    Example¶

    curl https://iwmi-leaf.onrender.com/api/gp/block/Digboi

    import requests

r = requests.get("https://iwmi-leaf.onrender.com/api/gp/block/Digboi")
data = r.json()
print(f"{data['block']} ({data['district']}): {len(data['gps'])} GPs")
for gp in data['gps']:
    print(f"  {gp['name']} — {gp['village_count']} villages")

    Get GP Locations¶

    Returns GP locations as both a flat list and grouped by block. Designed for populating dropdown menus.

    GET /api/gp/locations

    Response¶

    {
  "gps": [
    { "gp_name": "Rongagora", "gp_code": "12345", "block": "Digboi", "district": "Tinsukia" },
    { "gp_name": "Panitola", "gp_code": "12346", "block": "Digboi", "district": "Tinsukia" }
  ],
  "by_block": {
    "Digboi": [
      { "gp_name": "Rongagora", "gp_code": "12345", "block": "Digboi", "district": "Tinsukia" }
    ],
    "Doomdooma": [
      { "gp_name": "Rupaisiding", "gp_code": "12350", "block": "Doomdooma", "district": "Tinsukia" }
    ]
  }
}
    Field Type Description
    gps array Flat list of all GPs
    by_block object GPs grouped by block name (key: block name, value: array of GPs)

    Example¶

    curl https://iwmi-leaf.onrender.com/api/gp/locations

    const r = await fetch('/api/gp/locations');
const data = await r.json();

// Build grouped dropdown
Object.entries(data.by_block).forEach(([block, gps]) => {
  const group = document.createElement('optgroup');
  group.label = block;
  gps.forEach(gp => {
    const option = document.createElement('option');
    option.value = gp.gp_code;
    option.textContent = gp.gp_name;
    group.appendChild(option);
  });
  gpSelect.appendChild(group);
});

    Get GP Variables¶

    Returns metadata for all numeric variables available in the GP-level data, including data range statistics. Variable codes match the DSS metadata (same codes as block-level where applicable).

    GET /api/gp/variables

    Response¶

    [
  {
    "field": "AD",
    "label": "Agricultural Diversity",
    "description": "Percentage of agricultural land diversity",
    "group": "Land & Agriculture",
    "data_min": 0.0,
    "data_max": 95.5,
    "data_mean": 42.3
  },
  {
    "field": "BW",
    "label": "Number of Villages",
    "description": "Total number of villages in the GP",
    "group": "Other",
    "data_min": 1.0,
    "data_max": 25.0,
    "data_mean": 8.7
  }
]
    Field Type Description
    field string Column code in the GP data
    label string Human-readable name
    description string What this variable measures
    group string Category: Livestock, Transport & Connectivity, Finance & Markets, Utilities, Land & Agriculture, Activities, Other
    data_min number Minimum value across all GPs
    data_max number Maximum value across all GPs
    data_mean number Mean value across all GPs

    Errors¶

    Code Description
    404 GP data not available

    Example¶

    curl https://iwmi-leaf.onrender.com/api/gp/variables

    import requests

r = requests.get("https://iwmi-leaf.onrender.com/api/gp/variables")
variables = r.json()

# Group variables by category
by_group = {}
for v in variables:
    by_group.setdefault(v['group'], []).append(v)

for group, vars in by_group.items():
    print(f"\n{group}:")
    for v in vars:
        print(f"  {v['field']} — {v['label']} ({v['data_min']:.1f}–{v['data_max']:.1f})")

    Get GP Statistics¶

    Returns aggregate statistics for GP data including distribution across blocks.

    GET /api/gp/statistics

    Response¶

    {
  "total_gps": 95,
  "district": "Tinsukia",
  "blocks": {
    "Digboi": 12,
    "Doomdooma": 15,
    "Margherita": 10
  },
  "columns": ["GP_CODE", "GP_NAME", "Block_Name", "Dist_Name", "VIL_COUNT", "AD", "WA", "BW"]
}
    Field Type Description
    total_gps integer Total number of GPs
    district string District name (currently "Tinsukia")
    blocks object GP count per block
    columns array Available data columns

    Errors¶

    Code Description
    404 GP data not available

    Example¶

    curl https://iwmi-leaf.onrender.com/api/gp/statistics

    import requests

r = requests.get("https://iwmi-leaf.onrender.com/api/gp/statistics")
stats = r.json()
print(f"Total GPs: {stats['total_gps']}")
print(f"Blocks: {len(stats['blocks'])}")
for block, count in sorted(stats['blocks'].items()):
    print(f"  {block}: {count} GPs")
