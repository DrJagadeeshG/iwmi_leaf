    Locations API¶

    Hierarchical location data for districts, blocks, and Gram Panchayats. These endpoints are designed for populating dropdown menus, building navigation trees, and understanding the spatial hierarchy of the data.

    Get Location Hierarchy¶

    Returns the complete district → block → GP hierarchy. Districts with GP data are sorted first.

    GET /api/locations

    Response¶

    {
  "districts": [
    {
      "name": "Tinsukia",
      "has_gp_data": true,
      "blocks": [
        {
          "name": "Digboi",
          "gps": [
            { "name": "Rongagora", "code": "GP001" },
            { "name": "Panitola", "code": "GP002" }
          ]
        },
        {
          "name": "Doomdooma",
          "gps": [
            { "name": "Rupaisiding", "code": "GP010" }
          ]
        }
      ]
    },
    {
      "name": "Dibrugarh",
      "has_gp_data": false,
      "blocks": [
        { "name": "Barbaruah", "gps": [] },
        { "name": "Lahowal", "gps": [] }
      ]
    }
  ],
  "blocks": [
    { "district": "Tinsukia", "block_name": "Digboi" },
    { "district": "Tinsukia", "block_name": "Doomdooma" },
    { "district": "Dibrugarh", "block_name": "Barbaruah" }
  ]
}
    Field Type Description
    districts array Hierarchical district data, sorted with GP-enabled districts first
    districts[].name string District name
    districts[].has_gp_data boolean Whether GP-level data is available for this district
    districts[].blocks array Blocks in this district, each with a gps array
    blocks array Flat list of all blocks with their district (backward compatibility)

    Example¶

    curl https://iwmi-leaf.onrender.com/api/locations

    import requests

r = requests.get("https://iwmi-leaf.onrender.com/api/locations")
data = r.json()

for district in data['districts']:
    gp_label = " (GP data)" if district['has_gp_data'] else ""
    print(f"{district['name']}{gp_label}: {len(district['blocks'])} blocks")
    for block in district['blocks']:
        gp_count = len(block.get('gps', []))
        if gp_count:
            print(f"  └─ {block['name']}: {gp_count} GPs")

    const r = await fetch('/api/locations');
const data = await r.json();

// Build district dropdown
data.districts.forEach(d => {
  const option = document.createElement('option');
  option.value = d.name;
  option.textContent = d.name + (d.has_gp_data ? ' (GP)' : '');
  districtSelect.appendChild(option);
});

    Get All Districts¶

    Returns all districts with block counts and GP availability metadata.

    GET /api/districts

    Response¶

    {
  "districts": [
    {
      "name": "Tinsukia",
      "block_count": 8,
      "has_gp_data": true,
      "gp_count": 95
    },
    {
      "name": "Dibrugarh",
      "block_count": 7,
      "has_gp_data": false
    }
  ]
}
    Field Type Description
    districts[].name string District name
    districts[].block_count integer Number of blocks in this district
    districts[].has_gp_data boolean Whether GP data is available
    districts[].gp_count integer Number of GPs (only present when has_gp_data is true)

    Example¶

    curl https://iwmi-leaf.onrender.com/api/districts

    import requests

r = requests.get("https://iwmi-leaf.onrender.com/api/districts")
for d in r.json()['districts']:
    print(f"{d['name']}: {d['block_count']} blocks")

    Get District Boundaries¶

    Returns district boundary polygons as GeoJSON for map overlay display. These are the dashed outlines shown on the main map.

    GET /api/districts/geojson

    Response¶

    A GeoJSON FeatureCollection of district boundary polygons.

    {
  "type": "FeatureCollection",
  "features": [
    {
      "type": "Feature",
      "properties": {
        "dtname": "Tinsukia",
        "stname": "Assam"
      },
      "geometry": {
        "type": "Polygon",
        "coordinates": [[[95.3, 27.5], ...]]
      }
    }
  ]
}

    Errors¶

    Code Description
    404 District boundaries GeoJSON file not available

    Example¶

    curl https://iwmi-leaf.onrender.com/api/districts/geojson

    import requests
import geopandas as gpd

r = requests.get("https://iwmi-leaf.onrender.com/api/districts/geojson")
gdf = gpd.GeoDataFrame.from_features(r.json()['features'])
print(f"Districts: {len(gdf)}")

    const r = await fetch('/api/districts/geojson');
const geojson = await r.json();

// Add as dashed overlay on Leaflet map
L.geoJSON(geojson, {
  style: { fillColor: 'transparent', color: '#28537D', dashArray: '6,4' },
  interactive: false
}).addTo(map);

    Get Blocks in a District¶

    Returns all blocks belonging to a specific district.

    GET /api/districts/{district}/blocks

    Parameters¶

    Parameter In Type Required Description
    district path string Yes District name, case-sensitive (e.g. "Tinsukia")

    Response¶

    {
  "district": "Tinsukia",
  "blocks": [
    { "name": "Digboi", "district": "Tinsukia" },
    { "name": "Doomdooma", "district": "Tinsukia" },
    { "name": "Margherita", "district": "Tinsukia" }
  ]
}

    Errors¶

    Code Description
    404 District not found or has no blocks

    Example¶

    curl https://iwmi-leaf.onrender.com/api/districts/Tinsukia/blocks

    import requests

r = requests.get("https://iwmi-leaf.onrender.com/api/districts/Tinsukia/blocks")
data = r.json()
print(f"{data['district']}: {len(data['blocks'])} blocks")
for block in data['blocks']:
    print(f"  - {block['name']}")

    Deprecated Endpoints¶

    Deprecated

    The following endpoints are deprecated. They still work but may be removed in a future version.

    Deprecated Replacement Notes
    GET /api/district/names GET /api/districts Returns richer metadata (block counts, GP availability)
    GET /api/block/names GET /api/locations Returns full hierarchy instead of flat list
