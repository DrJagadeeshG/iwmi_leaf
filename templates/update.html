Handle With Care

This sheet contains raw data that powers the entire dashboard. Wrong edits break feasibility scores, maps, and analysis.

Mistake Impact
Editing BLOCK_ID or ID columns Block disappears or gets wrong data
Deleting a row Block vanishes from dashboard
Text in numeric column Shows "No data", breaks scores
Renaming column header Variable stops working everywhere
Empty cells "No data" for that block/variable
Duplicate BLOCK_ID Unpredictable results
New column added Won't show until added to Intervention Config sheet too
Safe columns to edit: Column F onwards (variable values). Never touch columns A–E (BLOCK_ID, STATE_ID, DISTRICT_I, id, Block_name).
Backup first: File > Download > CSV before bulk edits.
